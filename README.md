# Ugri Bugri

A lightweight, high-performance 2D platformer game developed with **Vanilla JavaScript** and **HTML5 Canvas**.
This project focuses on smooth physics and animations.

## 🚀 Key Features
- **Custom Physics Engine:** Realistic gravity, collision detection, and momentum-based movement.
- **Visual Polish:** Procedural grass animation (wind simulation) and a dynamic particle system for effects.
- **Performance Focused:** Includes a "Low Detail Mode" to ensure 60 FPS on older devices.
- **AI-Assisted Development:** Built using modern workflows, integrating AI for rapid prototyping and fine-tuning complex logic.

## 🛠️ Tech Stack
- HTML5 / CSS3
- JavaScript (ES6+)
- Canvas API

## 🕹️ How to Play
Use **Arrow Keys** or **WASD** to move and jump.
